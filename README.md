## 🎯 Learning Objectives
- Master the two-pointer approach for sorted arrays
- Solve pair/triplet sum problems in O(n) instead of O(n²)
- Apply fast-slow pointers on linked lists and arrays
- Implement Dutch National Flag (3-way partition)
- Handle duplicates correctly in two-pointer problems

## 📋 Problem Statements
### P1 — Two Sum Sorted: `[2,7,11,15],target=9→[1,2]`
### P2 — Three Sum: `[-1,0,1,2,-1,-4]→[[-1,-1,2],[-1,0,1]]`
### P3 — Container Water: `[1,8,6,2,5,4,8,3,7]→49`
### P4 — Sort Colors: `[2,0,2,1,1,0]→[0,0,1,1,2,2]`
### P5 — Remove Duplicates: `[1,1,2]→2 (in place)`
